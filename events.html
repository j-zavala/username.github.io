Haskell Workshop

Event Time and Date
- Wednesday, October 14 2015, 7:15 PM to 8:45 PM
Location:
- Business Building 129 1400 Washington Ave, Albany NY 12222

    About:

  • Haskell is a purely functional programming language. If you don't know what that means it's ok!
  • Haskell is unlike any other language they teach us in our classes. We'll be going over basic datatypes, statements, control flow (or lack there-of) and all of the awesome things that Haskell has to offer.
  • No programming experience needed! All you need is a laptop (or someone to look on with)!
  • If you want you can download the compiler, but we'll be starting off by using an online interpreter to make it less intimidating: https://www.haskell.org/ghc/
